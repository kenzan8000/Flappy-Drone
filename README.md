# Flappy Drone

Players control a drone and attempt to fly faster.


## Installation


### Server

```
$ git clone https://github.com/kenzan8000/Flappy-Drone.git
$ cd Flappy-Drone/server
$ npm install
$ npm start
```


### iOS

```
$ git clone https://github.com/kenzan8000/Flappy-Drone.git
$ cd Flappy-Drone/iOS
$ pod install
$ open Flappy-Drone.xcworkspace
```
